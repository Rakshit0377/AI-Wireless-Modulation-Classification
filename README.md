# AI-Based Wireless Signal Classification

## Overview
This project implements a Machine Learning system to automatically classify wireless signals based on their extracted signal features. The model learns patterns in signal characteristics and predicts the modulation type of incoming wireless signals.

Wireless signal classification is important in modern communication systems such as cognitive radio, spectrum monitoring, and intelligent receivers.
